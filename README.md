# Autonolas Protocol Frontend
This repository contains the [Autonolas protocol](https://protocol.autonolas.network/) frontend.

## Tech Stack
- web3.js
- ReactJS
- NextJS
- Styled-components
- Ant Design
- React-testing-library
- Jest

## Installation and Setup Instructions

Clone down this repository. You will need `node` version ">=14" and `yarn` version ">=1" installed globally on your machine.

Installation:

`yarn`

To Run Test Suite:

`yarn testc`

To Start Server:

`yarn dev`

To Visit App:

`localhost:3000`

Once you're running the app, you can connect to Valory's [Autonolas Registory](https://github.com/valory-xyz/autonolas-registries) through docker
